# 🛒 Shopping Page – Frontend Web Project
## 📌 Project Overview

This project is a **simple Shopping Page frontend application** built using **HTML, CSS, and JavaScript**.
It was developed as part of my **academic learning during Vihaan classes**, and it helped me understand the **fundamentals of web development**.

This is my **second project**, and it played an important role in strengthening my basics of **frontend design, structure, and user interface flow**.

---

## 🎯 Purpose of the Project

The main objective of this project was to:

* Understand **basic web development concepts**
* Learn how a **shopping website UI looks and behaves**
* Practice creating **multiple web pages** and linking them together
* Explore different **frontend features and tools**

This project focuses only on the **frontend** and does not include backend or database functionality.

---

## 🧰 Technologies Used

* **HTML** – Structure of the web pages
* **CSS** – Styling and layout design
* **JavaScript** – Basic form validation and interactivity

---

## 📄 Pages Included

* `shop.html` – Main shopping page
* `cart.html` – Cart page layout
* `log.html` – Login page
* `reg.html` – Registration page

---

## 🎨 Stylesheets

* `style1.css`
* `cart.css`
* `cat.css`

These files are used to style different pages and components of the application.

---

## ⚙️ JavaScript Functionality

* `valid.js` – Used for **basic form validation** and simple client-side logic
